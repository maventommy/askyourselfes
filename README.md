# Ask Yourselves

Talk to yourself, thirty years on.

Ask Yourselves is a mobile app (Expo / React Native, runs on iOS, Android, and web) where you meet an AI version of your future self: grounded in a short interview about your real life, shown to you as your own age-progressed face, answering in a voice you can play out loud. It is built on the finding from MIT Media Lab's peer-reviewed *Future You* research (2024) that a single conversation with a text-based future self measurably reduces anxiety and strengthens future-self continuity.

The brand stake: **the only AI you don't have to perform for.** No feed, no followers, no streaks to impress anyone with. Your future self can call BS on you, because there is no one else in the room.

## What it does

1. **Staged interview.** One question per screen while a progress bar fills: your name, your age, a moment you were proud of, what you are carrying, a decision you are weighing. The future self is forming as you answer.
2. **Choose your future.** A slider picks the age you want to talk to (30 to 80, five-year steps), with your current age as the baseline.
3. **Tone calibration.** Gentle, blunt, or playful. It changes how the future self talks to you.
4. **The reveal.** Add a selfie and meet the you who already lived it: your face, age-progressed to the age you chose, generated by OpenAI's gpt-image-2 image edit endpoint.
5. **The conversation.** Chat runs on OpenAI GPT-5.6 Luna, grounded in your interview and your message history. It remembers what you told it and it does not flatter.
6. **Hear it.** Every reply can be played out loud via OpenAI text to speech.

## Stack

- **Client:** Expo (React Native), TypeScript. One codebase for iOS, Android, and web.
- **Backend:** Supabase. Anonymous auth, Postgres with row-level security (every user sees only their own rows), Storage for portraits, and three Deno Edge Functions:
  - `future-self-chat`: builds the grounded system prompt, calls `gpt-5.6-luna`, persists the exchange.
  - `age-portrait`: selfie in, age-progressed portrait out via `gpt-image-2` (`/v1/images/edits`), stored per user, strictly one generation per user.
  - `speak`: text to speech for replies.
- **Models:** OpenAI GPT-5.6 Luna (chat), gpt-image-2 (age progression), gpt-4o-mini-tts (voice).
- **Unit economics:** roughly $0.002 per chat exchange, about $0.05 for the one-time portrait, about $0.001 per spoken reply.

## Run it

Prerequisites: Node 20+, a Supabase project, an OpenAI API key.

```bash
git clone <this repo>
cd ask-yourselves
npm install
```

1. Create a Supabase project. In the SQL editor, run `supabase/sql/schema.sql` (tables + RLS), then `supabase/sql/2026-07-18-portraits.sql` (portrait column + storage bucket). Enable anonymous sign-ins under Authentication settings.
2. Set the Edge Function secret: `supabase secrets set OPENAI_API_KEY=sk-... --project-ref <your-ref>`
3. Deploy the functions:
   ```bash
   supabase functions deploy future-self-chat --project-ref <your-ref>
   supabase functions deploy age-portrait --project-ref <your-ref>
   supabase functions deploy speak --project-ref <your-ref>
   ```
4. Create `.env.local` in the repo root:
   ```
   EXPO_PUBLIC_SUPABASE_URL=https://<your-ref>.supabase.co
   EXPO_PUBLIC_SUPABASE_ANON_KEY=<your publishable key>
   ```
5. Start it: `npx expo start --web` (or scan the QR with Expo Go for a phone).

## Tests

Prompt builders are unit tested with Deno:

```bash
deno test supabase/functions/future-self-chat/prompt_test.ts
deno test supabase/functions/age-portrait/prompt_test.ts
```

## Honest notes

- 18+ by policy. The interview and chat touch real feelings; this is not a toy for minors.
- The future self never claims to know your future. It speaks in earned perspective, not prophecy, and the system prompt forbids flattery.
- One aged portrait per user is enforced server-side, both as the free-tier spec and as a hard cost guard.
- The app was built with AI coding agents end to end; the product itself runs entirely on OpenAI models.
